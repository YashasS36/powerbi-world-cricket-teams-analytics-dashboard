# 🏏 Power BI | World Cricket Teams Analytics Dashboard

## 📌 Project Overview

This project is an interactive **Power BI dashboard** that analyzes world cricket teams based on performance metrics, leadership, trophies, and player statistics.
It transforms raw cricket data into meaningful insights using data visualization and storytelling techniques.

The dashboard helps users explore:

* Team performance (wins and losses)
* Trophy achievements
* Leadership impact (captains)
* Best players (batsmen, bowlers, fielders, finishers)
* Full Playing XI of each team


## 📊 Dataset Information

The project consists of two main tables:

### 1. Cricket Teams Table

Contains team-level performance data:

* Team Name
* Matches Won
* Matches Lost
* Trophies Won
* Best Captain
* Best Batsman
* Best Bowler
* Best Fielder
* Major Venues Played

### 2. Playing XI Table

Contains squad-level data:

* Team
* Captain
* Player 1 to Player 11


## 📈 Key Dashboards & Visuals

### 🏆 Team Victory Analysis

Displays total matches won by each team.
Helps identify the most successful teams based on performance.


### 🏅 Team Trophy Distribution

Shows number of trophies won by each team.
Highlights the most decorated cricket teams.


### 👨‍✈️ Team Performance & Leadership Analysis

Combines:

* Matches Won
* Matches Lost
* Trophies Won
* Best Captain

Provides insights into how leadership impacts team performance.


### ⭐ Team Performance by Leadership Analysis

Advanced visualization including:

* Matches Won
* Trophies Won
* Best Captain
* Best Batsman
* Best Bowler
* Win Percentage

Helps understand overall team strength and player contributions.


### 🌳 Treemap – Team Dominance

Represents team performance visually using size-based comparison of wins or trophies.


### 📋 Team Information Matrix

Displays detailed team information including:

* Captain
* Best Batsman
* Best Bowler
* Best Fielder

### 👥 Playing XI Matrix

Shows full playing XI of each team.
When a team is selected, the entire squad updates dynamically.


## 🔗 Interactivity Features

This dashboard is fully interactive:

* Click on any team to filter all visuals
* Slicers allow filtering by team, captain, and players
* Cross-visual interactions update KPIs dynamically
* Drill-down analysis for deeper insights
